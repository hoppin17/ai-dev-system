# AI_PROMPTS.md

이 문서는 AI 도구(GPT, Cursor 등)와 작업할 때 사용하는 **표준 프롬프트 모음**이다.
반복적으로 사용하는 프롬프트를 정리하여 일관된 작업 방식을 유지한다.

---

# 1. Conversation Handoff Prompt

대화 context가 길어졌을 때
**다음 대화로 넘어가기 위한 요약 문서**를 생성한다.

```
현재 대화의 context가 길어졌기 때문에 다음 대화로 넘어가기 위한
Conversation Handoff를 생성해줘.

목표:
다음 AI가 이 프로젝트 상황을 빠르게 이해하고 이어서 작업할 수 있도록
핵심 정보만 요약한다.

규칙:
- 불필요한 설명 제거
- 핵심만 정리
- 최대 500~800자 수준
- markdown 형식 사용
- 사실 기반 요약

다음 구조를 반드시 지켜서 작성해줘.

# Conversation Handoff

## 프로젝트 / 주제
현재 진행 중인 프로젝트나 논의 주제

## 현재까지 진행한 내용
이번 대화에서 완료된 작업

## 현재 상태
현재 프로젝트 또는 작업 상태

## 해결된 문제
이번 대화에서 해결한 문제

## 남은 문제
아직 해결되지 않은 문제

## 다음 대화에서 해야 할 작업
다음 AI가 이어서 진행해야 할 작업
```

---

# 2. Project Handoff Prompt

하루 작업이 끝났을 때
**프로젝트 진행 기록을 남기기 위한 문서**를 생성한다.

```
이번 대화를 기반으로 Project Handoff 문서를 생성해줘.

목표:
프로젝트 진행 상황과 의사결정을 기록하는 문서를 작성한다.

규칙:
- 작업 기록 중심
- 의사결정은 반드시 기록
- 불필요한 추측 금지
- markdown 형식 사용
- 가능한 구체적으로 작성

다음 구조를 반드시 지켜서 작성해줘.

# Project Handoff

작성일: YYYY-MM-DD

## 프로젝트
현재 프로젝트 이름 또는 주제

## 오늘 진행한 작업
오늘 완료한 작업 목록

## 주요 구현 / 설계 내용
코드 구조, 설계, 시스템 구조 등

## 해결한 문제
오늘 해결한 문제

## 주요 결정 사항
프로젝트에서 새롭게 결정된 정책 또는 설계

## 남은 작업
아직 구현되지 않은 작업

## 다음 작업 계획
다음 단계에서 진행할 작업
```
